# 📖 Overview

The **URL Shortener** is a simple web application built using **HTML**, **CSS**, and **JavaScript**. It allows users to enter a valid URL and generates a randomly created shortened link for easier sharing.

The application performs basic URL validation and provides options to generate a short URL or clear the input and output fields.

# 🖥️ User Interface

The application includes:

### 🔗 URL Input

- Text field for entering a long URL

### 🎛️ Buttons

- **Shorten URL** – Generates a random shortened URL
- **Clear** – Clears the input and generated result

### 📄 Result Section

Displays the generated short URL as a clickable link.

# 🔥 Calories Burnt Prediction using Machine Learning

## 📌 Project Overview

This project predicts the number of calories burned during physical activity using Machine Learning. The model is trained on a dataset containing user information such as age, gender, height, weight, exercise duration, heart rate, and body temperature.

The objective is to estimate calories burned based on physiological and exercise-related features.

---

## 📂 Dataset

The dataset contains the following features:

- Gender
- Age
- Height (cm)
- Weight (kg)
- Duration (minutes)
- Heart Rate (bpm)
- Body Temperature (°C)

**Target Variable**

- Calories Burned

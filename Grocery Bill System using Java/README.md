# 📖 Overview

The **Grocery Bill System** is a console-based Java application that generates a grocery bill based on user input. Users can enter multiple products along with their quantity and unit price. The program calculates the total cost for each product and displays a formatted bill summary with the overall amount.

# 📋 Bill Summary

The generated bill includes:

- 🛍️ Product Name
- 📦 Quantity
- 💵 Unit Price
- 🧾 Total Price per Product
- 💰 Grand Total

Example:

```text
********************************************* BILL SUMMARY *********************************************

Product        Quantity      Unit Price      Total

Rice              2           Rs50          Rs100
Milk              3           Rs30          Rs90
Sugar             1           Rs45          Rs45

Grand Total: Rs235
```

# 🌤️ Weather Application

A simple and responsive weather application built using **HTML**, **CSS**, and **JavaScript**. The application fetches real-time weather information using the **WeatherAPI** and displays the current weather conditions for any city entered by the user.

---

## 📖 Project Overview

The Weather Application allows users to search for a city and instantly view its current weather details. It provides a clean and modern user interface with a glassmorphism design and an animated background to enhance the user experience.

This project demonstrates the use of asynchronous JavaScript, API integration, DOM manipulation, and responsive frontend design.

---

## ✨ Features

- 🌍 Search weather by city name
- 🌡️ Displays current temperature in Celsius
- ☁️ Shows current weather condition
- 💧 Displays humidity percentage
- 🖼️ Weather condition icon
- 🎨 Modern glassmorphism user interface
- 🌈 Animated gradient background
- 🌙 Dark mode styling support
- ⚠️ Error handling for invalid city names and API failures

## 🚀 How It Works

1. Enter the name of a city.
2. Click the **Get Weather Details** button.
3. The application sends a request to the WeatherAPI.
4. The API returns the current weather data.
5. The application displays:
   - City and Country
   - Weather Condition
   - Weather Icon
   - Temperature (°C)
   - Humidity (%)

## 🔑 API Configuration

This project uses **WeatherAPI** to retrieve real-time weather information.

To use your own API key:

1. Create a free account on WeatherAPI.
2. Generate an API key.
3. Open `FrontPageArchitecture.js`.
4. Replace the existing API key with your own.

```javascript
const API_KEY = "YOUR_API_KEY";
```
